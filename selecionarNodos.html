METODO DESCRIPCION EJEMPLO
document.getElementById('id') Permite Acceder a un elemento a través de su id 1. document.getElementById('titulo')
1.1. const titulo = document.getElementById('titulo')
titulo.textContent = 'DOM - Accediendo a nodos'
document.getElementsByClassName('nombreDeClase') Permite Accede al primer elemento que coincida con el selector CSS, obtiene un array con todos los elementos hijos que tengan esa clase, ojo no podremos obtener solo 1 elemento con esa clase. 2. document.getElementsByClassName('border')
2.1. document.getElementsByClassName('fs-1')
document.getElementsByTagName('nombreDeEtiqueta') Permite Acceder al elemento por el nombre de la Etiqueta, document.getElementsByTagName('p') => con este método obtenemos una lista o “array list” con todos los elementos que tengan esa etiqueta, ejemplo todos los P. Al igual que con el método anterior no hay posibilidad de usarlo en singular. 3. document.getElementsByTagName('p')
3.1. document.getElementsByTagName('b')
document.querySelector('selector') Permite Acceder al primer elemento por medio de el selector, document.querySelector(' ') => nos permite buscar de 3 formas, con id, clase o tagName. A diferencia de los 2 anteriores este nos devuelve 1 solo elemento, el primero que contenga el valor que se le paso. Id => (’#id’), class => (’.border’), tagName (‘p’) 4. document.querySelector('.border')
4.1. document.querySelector('p')
4.2. document.querySelector('#titulo')
document.querySelectorAll('selector') Permite Acceder a todos los elementos por medio de el selector, document.querySelectorAll('') => este método retorna una array list con todos los elementos que tengan ese selector (id, class o tagName) Casi siempre el elemento “padre o parent” es document. ya que estamos haciendo referencia a todo el DOM, todo el documento y esto en ciertos casos nos permite evitar errores.
ejemplo = const button = document.querySelector(’#button) 		document.querySelectorAll('.paragraph')
