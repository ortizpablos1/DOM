ELIMINAR NODOS
METODO DESCRIPCION EJEMPLO
Remove Child Podemos eliminar un nodo según a un método que proviene del padre y pasamos la referencia a eliminar:
padre.removeChild(nodoAEliminar)
tambien podemos utilizar la propiedad parentElement 		// seleccionar le nodo a eliminar
const nodoAEliminar = document.querySelector('h3');
// Seleccionar el padre directo del elemento a eliminar
const padre = document.querySelector('div');
//Eliminar el Nodo
padre.removeChild(nodoAEliminar);
// propiedad parentElement
const nodoAEliminar = document.querySelector('h3');
nodoAEliminar.parentElement.removeChild(nodoAEliminar);
Remove Es la evolución de remove child pero No esta soportado en internet explorer // eliminar directamente
document.querySelector('a').remove()
// 1. Seleccionar el nodo a eliminar
const eli = document.querySelector('.gb_d')
// 2. Eliminar nodo seleccionado
eli.remove();
Replace Child Nos permite remplazar un nodo
padre.replaceChild(nuevoNodo, aRemplazar) 		// 1. seleccionar el padre del elemento a reemplazar
const padre = document.querySelector('center');
// 2. Seleccionar elemento hijo a reeemplazar
const areemplazar = document.querySelector('.gNO89b');
// 3. crear nuevo elemento para reemplazar
const nuevo = document.createElement('h2');
// 4. crear contenido para el h2 creado
nuevo.textContent = "GoogleBusy";
// 5. Realizar el reemplazo
padre.replaceChild(nuevo, areemplazar);
