CREAR NODOS
METODO DESCRIPCION EJEMPLO
document.createElement(' ') Mediante el método .createElement() podemos crear un elemento HTML en memoria (¡no estará insertado aún en nuestro documento HTML!). Con dicho elemento almacenado en una variable, podremos modificar sus características o contenido, para posteriormente insertarlo en una posición determinada del DOM o documento HTML. Creamos un Elemento DIV
const div = document.createElement("div");
document.createTextNode("Texto") Permite Crear nodos de texto sin etiqueta HTML, pasándole a ambos un String con el texto en cuestión. En ambos, se devuelve un Node que podremos utilizar luego para insertar en el documento HTML Crear un nodo de texto con la palabra Hola Mundo
const text = document.createTextNode("Hola Mundo");
document.createComment('Texto de comentario') Permite crear comentarios HTML con createComment() Crear comenario para el codigo
const comment = document.createComment("Comentario");
INSERTAR NODOS
METODO DESCRIPCION EJEMPLO
appendChild Este método nos permite agregar un elemento (Nodo) al final de la lista de hijos de un elemento padre previamente especificado, es decir, agrega un elemento dentro de un padre, y este es colocado al final. Es importante resaltar que si el Nodo que se esta insertando existe en el documento actual, es una referencia, este sera removido del padre actual y sera puesto dentro del padre que se esta especificando. por otro lado este método retorna o devuelve el elemento Nodo insertado. Supongamos que tenemos una lista, y queremos insertar un nuevo elemento con el texto “Item 3”, lo haremos de la siguiente manera // Seleccionar el Luar donde va el nuevo Nodo
const container = document.querySelector("#contain");
// Crear el nodo o elemento a agregar
const h3 = document.createElement("h3");
// Agregar el nodo al ugar seleccionado
container.appendChild(h3) // Crear un texto para insertarlo al nodo creado const text = document.createTextNode("Hola Mundo");
// Insertamos el texto dentro del item
h3.appendChild(text);
append Este método es una evolución de appendChild, donde podemos agregar más de un nodo, cosa que no podemos hacer en el método anterior, podemos agregar texto. Este también agrega los nodos al final de los hijos del elemento padre especificado, y no retorna ningún elemento. No tiene soporte adecuado para IE 11. const container = document.querySelector("ul.container");
const item = document.createElement("li");
const subItem = document.createElement("ul");
// Agregamos el texto de item, adicional insertamos la nueva lista
item.append("Categorias", subItem);
// Agremos el item al menu
container.append(item);
// modo inline
conainer.append('Texto', document.createElement('div'))
insertBefore Este método permite insertar o agregar un nodo, antes del nodo que pasemos de referencia, y que pertenece al padre del que estamos accediendo a dicho método. Si insertamos un Nodo que ya esta en nuestro DOM el método lo cambia de lugar, es decir, lo quita de donde se encuentra actualmente y lo inserta en el sitio al que estamos haciendo referencia. Este método recibe dos argumentos, el primero de ellos es el Nodo que queremos insertar, y el segundo es el Nodo de referencia. // Seleccionar el Luar donde va el nuevo Nodo
const container = document.querySelector("#contain");
// Crear el nodo o elemento a agregar
const h3 = document.createElement("h3");
// Crear la referencia (debe ser elemento siguiente)
const referencia = document.querySelector(".titulo"); // Agregar el nodo antes de la referencia
container.insertBefore(h3, referencia)
insertAdjacentElement Este método permite insertar un nodo de acuerdo a una posición especificada, y lo ubica con respecto al elemento sobre el cual es llamado. Es el que más opciones de inserción nos proporciona. Este método recibe 2 argumentos, el primero de ellos donde se insertará el nuevo nodo, para esto contamos con cuatro opciones, las cuales son:
beforebegin: Inserta el elemento antes de la referencia. Es decir, antes del nodo que llama al método en cuestión.
afterbegin: Insertar el nodo dentro del elemento referencia, pero ademas de esto lo inserta antes de su primer hijo.
beforeend: Al igual que la opción anterior lo agrega dentro del elemento referencia, con la particularidad de que ahora lo inserta después de su ultimo hijo.
afterend: Inserta el elemento después del nodo referencia, es decir, despues del nodo que llama al método. 		// Seleccionar el Luar donde va el nuevo Nodo
const container = document.querySelector("#contain");
// Crear la referencia (debe ser elemento siguiente)
const referencia = document.querySelector(".titulo"); // Crear el nodo o elemento a agregar
const h3 = document.createElement("h3");
// Agregar el nodo segun la referencia
referencia.insertAdjacentElement('beforebegin', h3)
referencia.insertAdjacentElement('afterbegin', h3)
referencia.insertAdjacentElement('beforeend', h3)
referencia.insertAdjacentElement('afterend', h3)
